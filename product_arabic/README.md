# Arabic Product Name in POS - Odoo 19

This module adds Arabic product name support to the Point of Sale (POS) in Odoo 19.

## Features

- Add Arabic names to products
- Display Arabic names in POS product screen
- Show Arabic names in POS order lines
- Include Arabic names in POS receipts
- Toggle Arabic display from POS configuration

## Installation

1. Copy the `product_arabic` folder to your Odoo addons directory
2. Update your addons list in Odoo
3. Install the module from Apps menu

## Configuration

1. Go to Point of Sale > Configuration > POS
2. Enable "Show Arabic Product Names in POS" option
3. Go to Products and add Arabic names to your products

## Usage

1. Add Arabic name to any product in the product form
2. Open POS and the Arabic name will be displayed alongside the regular name
3. Arabic names will also appear on receipts

## Compatibility

- Odoo 19 Community
- Odoo 19 Enterprise
- Odoo.sh
