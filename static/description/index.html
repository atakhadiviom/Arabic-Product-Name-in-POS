Arabic Product Name in POS

Display Arabic Product Names in POS Screen and Receipts

Key Features

  • Add Arabic names to your products
  • Display Arabic names in POS product screen
  • Show Arabic names in POS order lines
  • Include Arabic names in POS receipts
  • Easy to configure - just add Arabic name to product
  • Works with Odoo 19 Community, Enterprise & Odoo.sh

How to Use

1. Enable Arabic Product Names:
Go to POS Configuration and enable "Show Arabic Product Names in POS" option.

2. Add Arabic Names to Products:
Go to Products > select a product > add the Arabic name in the "Arabic Name" field.

3. Use POS:
Open your POS and you will see the Arabic names displayed for products that have them.

Technical Details

  • Compatible with Odoo 19
  • Works with both Community and Enterprise editions
  • Supports Odoo.sh
  • Uses Odoo's built-in translation mechanism
  • RTL (Right-to-Left) support for Arabic text
